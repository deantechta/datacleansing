# Excel Column Picker

엑셀(xlsx/xls) 및 CSV 파일에서 원하는 컬럼 정보를 빠르게 추출해 클립보드에 복사하는 **순수 프론트엔드 웹 도구**입니다.
서버 업로드 없이 브라우저에서 완전히 동작합니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 복수 파일 지원 | xlsx, xls, csv 파일을 여러 개 동시 업로드 |
| 복수 시트 선택 | 파일 내 여러 시트를 동시에 처리 |
| 헤더 행 지정 | 시트별로 컬럼명이 있는 행 번호를 자유롭게 설정 |
| 컬럼 선택 | 카드 UI로 원하는 컬럼만 골라서 선택 |
| 클립보드 복사 | 선택한 컬럼을 `열문자: 컬럼명` 형식으로 복사 |
| 파일명 표시 | 복사 결과 최상단에 파일명(시트명) 자동 포함 |

---

## 사용 방법

1. **파일 업로드** — 파일을 드래그하거나 클릭해서 선택 (복수 선택 가능)
2. **시트 선택** — 처리할 시트를 하나 이상 선택
3. **헤더 행 설정** — 각 시트의 컬럼명이 위치한 행 번호 입력 (기본값: 1행)
4. **컬럼 선택** — 추출할 컬럼 카드를 클릭해서 선택
5. **클립보드 복사** — 결과를 확인하고 복사 버튼 클릭

---

## 복사 결과 형식

### 단일 파일 · 단일 시트
```
[data.xlsx]
A열: 이름
B열: 나이
C열: 이메일
```

### 단일 파일 · 복수 시트
```
[data.xlsx / 고객정보]
A열: 이름
B열: 연락처

[data.xlsx / 주문내역]
A열: 주문번호
B열: 상품명
```

### 복수 파일
```
[고객.xlsx / Sheet1]
A열: 고객명
B열: 등급

[주문.csv / Sheet1]
A열: 주문ID
C열: 금액
```

---

## 지원 파일 형식

- `.xlsx` — Excel 2007 이상
- `.xls` — Excel 97~2003
- `.csv` — UTF-8 인코딩 권장

---

## 기술 스택

- **순수 HTML/CSS/JavaScript** — 별도 빌드 도구 불필요
- **[SheetJS (xlsx)](https://sheetjs.com/)** — 엑셀/CSV 파싱 라이브러리 (CDN)

---

## 보안

- 파일은 브라우저 메모리에서만 처리되며 외부 서버로 전송되지 않습니다.
- 파일명·시트명·컬럼명은 모두 DOM `textContent`로 안전하게 처리되어 XSS를 방지합니다.
