# Excel Column Picker

엑셀(xlsx/xls) 및 CSV 파일에서 원하는 컬럼 정보를 빠르게 추출해 클립보드에 복사하는 **순수 프론트엔드 웹 도구**입니다.
서버 업로드 없이 브라우저에서 완전히 동작합니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 복수 파일 지원 | xlsx, xls, csv 파일을 여러 개 동시 업로드 (최대 50MB) |
| 복수 시트 선택 | 파일 내 여러 시트를 동시에 처리 |
| 시트 미리보기 | 시트 선택 단계에서 첫 5행을 테이블로 미리보기 |
| 헤더 행 지정 | 시트별로 컬럼명이 있는 행 번호를 자유롭게 설정 |
| 컬럼 선택 | 카드 UI로 원하는 컬럼만 골라서 선택 |
| 공통 컬럼 탐지 | 복수 시트 선택 시 2개 이상 시트에 동일 이름으로 존재하는 컬럼에 뱃지 표시 및 일괄 선택 |
| 컬럼 검색 | 컬럼 이름 또는 열 문자로 실시간 필터링 |
| 컬럼 카드 복사 버튼 | 각 컬럼 카드의 copy 버튼 클릭 시 "C열 총원" 형태로 클립보드에 복사 |
| 컬럼 미리보기 | 컬럼 카드에 마우스를 올리면 첫 5개 데이터 값 tooltip 표시 |
| 클립보드 복사 | 탭 / CSV / 줄바꿈 / LLM 마크다운 형식 선택 후 클립보드에 복사 |
| 프롬프트 작성 | 컬럼 선택 단계에서 분석 요청사항 작성, 파일명·컬럼 칩 클릭으로 프롬프트에 삽입 |
| 프롬프트 템플릿 | 중복 검사, 결측값 분석, 데이터 비교, 요약 통계, 데이터 정제 등 구체적 템플릿 제공 |
| 통합 복사 | 프롬프트 + 데이터 마크다운을 합쳐서 클립보드에 한번에 복사 |

---

## 사용 방법

1. **파일 업로드** — 파일을 드래그하거나 클릭해서 선택 (복수 선택 가능)
2. **시트 선택** — 처리할 시트를 하나 이상 선택 (미리보기로 내용 확인 가능)
3. **헤더 행 설정** — 각 시트의 컬럼명이 위치한 행 번호 입력 (기본값: 1행)
4. **컬럼 선택 및 프롬프트** — 추출할 컬럼 카드를 클릭해서 선택 + 분석 요청사항 작성 (템플릿 제공, 파일명/컬럼 칩 클릭으로 삽입)
5. **결과 확인 및 복사** — 포맷 선택 후 클립보드에 복사 (분석 요청사항이 자동으로 함께 복사됨)

## 복사 형식

### 줄바꿈 모드
```
[data.xlsx]
A열: 이름
B열: 나이
C열: 이메일
```

### 탭 모드 (Excel 붙여넣기용)
```
A열	B열	C열
이름	나이	이메일
```

### CSV 모드
```
A열,B열,C열
이름,나이,이메일
```

### 복수 시트 선택 시
```
[data.xlsx / 고객정보]
A열: 이름
B열: 연락처

[data.xlsx / 주문내역]
A열: 주문번호
B열: 상품명
```

### LLM 마크다운 모드
복수 파일/시트를 LLM에게 분석 요청할 때 사용합니다. 실제 파일명을 사용하여 공통 컬럼·전체 컬럼 수·열 위치를 함께 출력하여 LLM이 데이터 구조를 정확하게 파악할 수 있도록 합니다.

```markdown
## 데이터 구조

### 공통 컬럼 (분석 기준 Key)
| 컬럼명 | 포함 테이블 |
|--------|------------|
| customer_id | sales.csv (A열), customers.xlsx / 고객정보 (B열) |
| region | sales.csv (E열), customers.xlsx / 고객정보 (D열) |

### 테이블 목록
| 파일명 | 시트 | 전체 컬럼 수 |
|--------|------|-------------|
| sales.csv | — | 8 |
| customers.xlsx | 고객정보 | 6 |

### 분석 필요 컬럼
**sales.csv**
`A열: customer_id`, `C열: order_date`, `E열: region`

**customers.xlsx / 고객정보**
`B열: customer_id`, `C열: name`, `D열: region`
```

**프롬프트 작성 기능:** 컬럼 선택 단계(Step 4)에서 전체 컬럼을 보면서 분석 요청사항을 작성할 수 있습니다. 5종의 구체적 템플릿(중복 검사, 결측값 분석, 데이터 비교, 요약 통계, 데이터 정제)을 제공하며, 파일명·컬럼 칩을 클릭해 프롬프트에 삽입합니다. 복사 시 프롬프트와 데이터 마크다운이 합쳐져 LLM 채팅에 바로 붙여넣을 수 있습니다.

**활용 예시:**
> 다음 데이터에서 `A열: customer_id`를 기준으로 두 테이블을 연결하고, `C열: order_date`별 매출 합계를 분석해줘.

**특징:**
- **실제 파일명 사용**: 별칭 없이 파일명으로 직관적 참조
- **열 위치 표시**: 같은 이름의 공통 컬럼도 테이블마다 위치(열)가 다를 수 있어 명확히 구분
- **전체 컬럼 수**: 선택한 컬럼 수가 아닌 시트 전체 컬럼 수를 표시해 LLM이 데이터 규모를 정확히 인식
- **공통 컬럼 자동 탐지**: 선택한 컬럼 중 2개 이상 시트에 동일 이름으로 존재하면 자동으로 Key 섹션에 표시

## 지원 파일 형식

- `.xlsx` — Excel 2007 이상
- `.xls` — Excel 97~2003
- `.csv` — UTF-8 인코딩 권장

## 기술 스택

- **순수 HTML/CSS/JavaScript** — 별도 빌드 도구 불필요
- **[SheetJS (xlsx)](https://sheetjs.com/)** — 엑셀/CSV 파싱 라이브러리 (CDN)

---

## 보안

- 파일은 브라우저 메모리에서만 처리되며 외부 서버로 전송되지 않습니다.
- 파일명·시트명·컬럼명은 모두 DOM `textContent`로 안전하게 처리되어 XSS를 방지합니다.
- 파일 크기는 50MB로 제한됩니다.

```bash
# 또는 간단한 로컬 서버로 실행
npx serve .
```
