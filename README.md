# Excel Column Picker

엑셀(xlsx/xls) 및 CSV 파일에서 원하는 컬럼 정보를 빠르게 추출해 클립보드에 복사하는 **순수 프론트엔드 웹 도구**입니다.
서버 업로드 없이 브라우저에서 완전히 동작합니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 복수 파일 지원 | xlsx, xls, csv 파일을 여러 개 동시 업로드 (최대 50MB) |
| 복수 시트 선택 | 파일 내 여러 시트를 동시에 처리 |
| 시트 미리보기 | 시트 선택 단계에서 첫 5행을 테이블로 미리보기 |
| 헤더 행 지정 | 시트별로 컬럼명이 있는 행 번호를 자유롭게 설정 |
| 컬럼 선택 | 카드 UI로 원하는 컬럼만 골라서 선택 |
| 컬럼 검색 | 컬럼 이름 또는 열 문자로 실시간 필터링 |
| 컬럼 미리보기 | 컬럼 카드에 마우스를 올리면 첫 5개 데이터 값 tooltip 표시 |
| 클립보드 복사 | 탭 / CSV / 줄바꿈 구분자 선택 후 클립보드에 복사 |

---

## 사용 방법

1. **파일 업로드** — 파일을 드래그하거나 클릭해서 선택 (복수 선택 가능)
2. **시트 선택** — 처리할 시트를 하나 이상 선택 (미리보기로 내용 확인 가능)
3. **헤더 행 설정** — 각 시트의 컬럼명이 위치한 행 번호 입력 (기본값: 1행)
4. **컬럼 선택** — 추출할 컬럼 카드를 클릭해서 선택 (검색 필터 지원)
5. **클립보드 복사** — 구분자 형식을 선택하고 복사 버튼 클릭

---

## 복사 형식

### 줄바꿈 모드
```
[data.xlsx]
A열: 이름
B열: 나이
C열: 이메일
```

### 탭 모드 (Excel 붙여넣기용)
```
A열	B열	C열
이름	나이	이메일
```

### CSV 모드
```
A열,B열,C열
이름,나이,이메일
```

### 복수 시트 선택 시
```
[data.xlsx / 고객정보]
A열: 이름
B열: 연락처

[data.xlsx / 주문내역]
A열: 주문번호
B열: 상품명
```

---

## 지원 파일 형식

- `.xlsx` — Excel 2007 이상
- `.xls` — Excel 97~2003
- `.csv` — UTF-8 인코딩 권장

---

## 기술 스택

- **순수 HTML/CSS/JavaScript** — 별도 빌드 도구 불필요
- **[SheetJS (xlsx)](https://sheetjs.com/)** — 엑셀/CSV 파싱 라이브러리 (CDN)

---

## 보안

- 파일은 브라우저 메모리에서만 처리되며 외부 서버로 전송되지 않습니다.
- 파일명·시트명·컬럼명은 모두 DOM `textContent`로 안전하게 처리되어 XSS를 방지합니다.
- 파일 크기는 50MB로 제한됩니다.

---

별도 서버 없이 `index.html`을 브라우저에서 바로 열면 됩니다.

# 또는 간단한 로컬 서버로 실행
